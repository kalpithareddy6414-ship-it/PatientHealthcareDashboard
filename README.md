# Healthcare Risk Analysis Dashboard

A Tableau-based data visualization project analyzing patient health data 
to identify risk patterns using heart rate, age, and diagnosis.

##  Features
- Heart Rate vs Risk Score analysis
- Risk categorization (Low, Moderate, High)
- Age-based visualization using size encoding
- Ward and diagnosis insights

##  Insights
- High-risk patients have significantly higher risk scores
- Certain wards show higher concentration of critical patients
- Age contributes to increased risk in several cases
